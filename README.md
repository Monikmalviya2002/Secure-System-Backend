##  Overview

This is the **backend** for the **Secure Incident Reporting System**, a role-based incident management platform.  

It provides REST APIs for:

- **Users:** Create and manage their incidents.
- **Admins:** Manage all incidents, view analytics, and update statuses.
- **Super Admins:** Manage users, assign roles, block/unblock users, and view audit logs.

Built with **Node.js**, **Express**, and **MongoDB**.

---

##  Technologies Used

- **Node.js**  
- **Express.js**  
- **MongoDB & Mongoose**  
- **JWT / Session-based Authentication**  
- **bcrypt** for password hashing  
- **dotenv** for environment variables
